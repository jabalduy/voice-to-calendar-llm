# Voice to Calendar LLM

Pet-проект для демонстрации автоматизации добавления событий
в Google Calendar с помощью голосовых сообщений в Telegram
и обработки текста нейросетью.

## Идея проекта
Пользователь отправляет голосовое или текстовое сообщение в Telegram,
например:  
"Созвон с командой завтра в 18:30".

Система автоматически:
1. Преобразует голос в текст
2. Понимает смысл сообщения (задачу, дату, время)
3. Добавляет событие в Google Calendar

## Как реализовано
Вся логика автоматизации реализована в n8n.
Нейросеть используется для обработки естественного языка
и понимания обычных человеческих формулировок
(например: "завтра", "вечером", "после обеда").

## Для кого этот проект
Проект создан как демонстрационный pet-проект
для изучения прикладного ML и LLM.
