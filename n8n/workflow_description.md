# n8n Workflow Description

## Общая логика
Workflow принимает сообщения из Telegram (текст или голос)
и автоматически добавляет события в Google Calendar.

## Шаги workflow

1. **Telegram Trigger**  
   Получает входящее сообщение от пользователя.
   Поддерживаются текстовые и голосовые сообщения.

2. **Speech-to-Text (Whisper)**  
   Если сообщение голосовое, оно преобразуется в текст.

3. **LLM Processing**  
   Нейросеть анализирует текст и извлекает:
   - название события
   - дату
   - время  
   Поддерживаются естественные формулировки
   (например: "завтра", "вечером", "после обеда").

4. **Data Validation**  
   Проверка корректности распознанных данных
   перед созданием события.

5. **Google Calendar**  
   Создание события в календаре пользователя.
